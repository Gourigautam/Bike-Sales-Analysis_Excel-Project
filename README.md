# Retail Pricing Analysis using SQL

# Overview

This project contains data analysis performed using Microsoft Excel on a bike buyers dataset.

# Objectives

* Analyze customer purchasing behavior
* Identify factors affecting bike purchases
* Create interactive dashboards and visualizations

# Tools Used

* Microsoft Excel
* Pivot Tables
* Charts
* Slicers
* Data Cleaning Techniques

# Dataset Features

* Age
* Gender
* Income
* Education
* Occupation
* Region
* Marital Status
* Bike Purchase Status

# Analysis Performed

* Income vs Bike Purchase
* Age Group Analysis
* Region-wise Analysis
* Customer Demographics Visualization

# Skills Demonstrated

* Data Cleaning
* Data Visualization
* Dashboard Creation
* Business Insight Generation
* Excel Analytics
